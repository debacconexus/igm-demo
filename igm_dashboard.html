How to Read System Health
What Each Module Does
auth
API key validation. Confirms that every inference request carries a valid, authorized key before processing begins.
gate
Language governance matrix. Loaded with 23 supported languages — enforces domain classification and language routing before inference.
metering
Governance unit tracking. Records every inference for billing, accountability, and compliance reporting in real time.
nerp
Token budget enforcement. Sets and enforces energy-aware token ceilings per inference — prevents runaway AI output.
ttp
Energy telemetry monitor. Measures computational energy (joules) consumed per inference using the reference environment power sampler.
output_gateway
Cryptographic signing engine. Every governed inference output is signed with an Ed25519 key and assigned an immutable catalogue ID.
zitg
Zero-Inference Trust Governance. Verifies model integrity at startup. Requires local Ollama on the reference environment — shows degraded on cloud deployment.
ollama
Local LLM engine. Used for local governed inference on the reference iMac. Shows unreachable on cloud deployments — inference routes to Anthropic instead.
catalogue entries
Total cryptographically signed inference records stored in the immutable output catalogue since deployment.
active keys
Number of API keys currently authorized to submit governed inferences to the pipeline.
Status Meanings
ok — Module fully operational. No action required.
degraded — Running in fallback mode. Core governance still active but one component is unavailable. Expected on cloud deployments.
unreachable — Component is offline. For cloud deployments, ollama unreachable is expected and normal — inference routes to Anthropic.
mock — Running in test mode. Real inference is disabled. Not used in production.
About This System
The IGM pipeline runs on a three-tier architecture: inference governance (GATE + NERP + ZITG), energy accountability (TTP), and cryptographic provenance (OutputGateway). Every governed inference passes through all active modules before a response is released.
